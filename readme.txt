== Description ==

<strong>What is Blog Toplist?</strong>

Member of your website can add their web or blog to be listing at your blog with alexa, technorati and pagerank rangking.

Automatic get title and descriptions for web/blog.

<strong>Current version.</strong>

At this moment, only alexa ranking to be add for web/blog.

== Installation ==

This section describes how to install the plugin and get it working.

= Manual Installation =
1. Upload `blog-toplist` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Blog Toplist menu to get shortcode and function.

= Auto Installation =
1. Click Add New at Plugins Admin menu
2. Search Plugins = blog-toplist
3. Click Detail for plugins infomation and click Install
